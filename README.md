## How to Use?



1. Download the Project   Zip

2. Find out your VS folder/directory.
    - copy the .glut.h to include folder
    - copy the opengl.h to include folder.
    - copy the glut.dll to system32
    
3. Open the Project, and press F5 to Run 
   

**Enjoy!!**
